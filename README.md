# Microwave PC LCD
Anything related to the LCD on the control panel of baid's [custom microwave PC build](https://imgur.com/a/Ty57pLM). 
![](https://github.com/Git-baid/Microwave-PC-LCD/blob/main/20230725_234915.jpg)
LCD is controlled by an ESP32 microcontroller

Displaying an image is interfaced with [baidbot discord bot](https://github.com/Git-baid/baidbotDiscord)
